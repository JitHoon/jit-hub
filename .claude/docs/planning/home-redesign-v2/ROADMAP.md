# 로드맵: 홈페이지 리디자인 v2 (세로 스크롤 구조)

## MVP -- 즉시 구현

목표: 상단 헤더(로고 + Projects + 테마토글) + 히어로 그래프(풀 뷰포트) + 노드 클릭 시 슬라이드 다운 콘텐츠 펼침의 세로 스크롤 구조 완성.

- [ ] SiteHeader 컴포넌트 생성 (sticky top, 로고/Projects/ThemeToggle)
- [ ] ThemeToggle을 fixed에서 헤더 내부로 이동
- [ ] HomeLayout을 좌우 분할에서 세로 스크롤 구조로 전면 재작성
- [ ] 그래프 영역을 `calc(100vh - 56px)` 높이의 히어로 섹션으로 변경
- [ ] 노드 클릭 시 콘텐츠 슬라이드 다운 애니메이션 + 자동 스크롤
- [ ] 다른 노드 클릭 시 콘텐츠 교체 + 본문 상단 스크롤
- [ ] URL `?node=slug` 동기화 (기존 로직 유지)
- [ ] 기존 좌우 분할 컴포넌트 정리 (ContentPanelWrapper, CloseButton 등)
- [ ] `/nodes/[slug]` SEO 정적 페이지 생성
- [ ] 반응형 대응 (데스크톱/모바일 동일 구조)
- [ ] 접근성 (aria-label, 키보드 탐색)

예상 리소스: 개발 5일, 디자인 1일

## Phase 2 -- 검증 후

조건: MVP 배포 후 사용자가 화면을 직접 확인하고 만족할 때 진행.

- [ ] `/projects` 페이지 구현 (외부 링크 카드 리스트)
- [ ] 콘텐츠 접기 기능 (그래프만 보기로 복귀)
- [ ] 콘텐츠 영역 내 "관련 노드" 클릭 시 해당 노드로 전환
- [ ] 그래프 카메라 힌트 UI (첫 진입 안내)

## Phase 3 -- 미래

아직 구체화하지 않을 기능들:
- 노드 검색/필터 (클러스터별, 키워드)
- 헤더에 추가 내비게이션 (Articles 등)
- 다국어(i18n) 지원
- 방문자 Analytics 통합
- 그래프 인터랙션 강화 (연결선 호버 하이라이트, 경로 시각화)
