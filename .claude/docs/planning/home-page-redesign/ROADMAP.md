# 로드맵: 홈페이지 리디자인 (Starfield Graph 3D)

## MVP -- 즉시 구현

목표: 홈 접속 시 노드가 3D 공간에서 빛나며 떠다니고, 카메라를 회전/줌하며 탐색하고, 클릭 시 분할 뷰로 콘텐츠를 확인할 수 있는 최소 완성 경험.

- [ ] Three.js 의존성 설치 (three, @types/three, react-force-graph-3d)
- [ ] GraphCanvas3D 기본 3D 렌더링 (풀스크린, 클러스터 색상 노드)
- [ ] 카메라 제어 (자동 회전, 드래그 회전, 줌, 노드 포커스 이동)
- [ ] 별빛(Starfield) 3D emissive glow 효과 + Fog 깊이감
- [ ] 노드 선택 + URL `?node=slug` 동기화
- [ ] 풀스크린 <-> 분할 뷰 전환 애니메이션 (3D 씬 리사이즈 포함)
- [ ] ContentPanel (MDX 렌더링 + 기존 컴포넌트 패턴 조합)
- [ ] CameraHint (조작 안내 UI) + WebGL 폴백 메시지
- [ ] 반응형 대응 (Desktop/Mobile)
- [ ] 접근성 (키보드 탐색, aria-label, 숨겨진 노드 목록)
- [ ] 성능 최적화 (60fps 확인, 모바일 LOD)

예상 리소스: 개발 10일, 디자인 3일

## Phase 2 -- 검증 후

조건: MVP 배포 후 실제 방문자 데이터 확인. 노드 클릭률 30% 이상, 3D 인터랙션 비율 40% 이상이면 진행.

- [ ] 상단 네비게이션 바 (`/`, `/article`, `/project` 라우팅)
- [ ] 노드 간 연결선 호버 인터랙션 (3D 공간에서 연결 노드 하이라이트)
- [ ] 클러스터별 필터 UI (사이드바 또는 토글 버튼)
- [ ] 콘텐츠 패널 내 "관련 노드" 클릭 시 카메라 포커스 이동
- [ ] 카메라 프리셋 뷰 (클러스터별 최적 카메라 앵글)

## Phase 3 -- 미래

아직 구체화하지 않을 기능들:
- 노드 검색 (키워드 기반)
- 3D 공간 파티클 효과 (배경 별 파티클)
- 노드 추가 알림 (새 콘텐츠 표시)
- `/article` 페이지 (시계열 블로그 형식)
- `/project` 페이지 (프로젝트 카드 갤러리)
- Analytics 대시보드 통합
- 영문 버전 (i18n)
- VR/AR 모드 (Three.js WebXR 확장)
