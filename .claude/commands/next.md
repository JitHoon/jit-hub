## 지시사항

`progress.md`를 읽고 다음 미완료 작업 **딱 하나**를 찾아서 실행하라.

## 프로세스

### 1단계: 다음 작업 식별

1. `progress.md`를 읽어라
2. 체크리스트에서 첫 번째 `- [ ]` 항목을 찾아라 (위에서 아래 순서)
3. 해당 항목이 속한 Phase와 브랜치를 확인하라
4. 의존성 확인: 해당 Phase의 이전 항목이 모두 `[x]`인지 검증하라

### 2단계: 작업 실행

1. 식별된 **하나의 작업만** 수행하라
2. 해당 Phase에 명시된 브랜치가 있으면 브랜치 상태를 확인하라
3. 작업을 완료하라
4. 해당 Phase의 검증 기준이 있으면 검증하라

### 3단계: progress.md 업데이트

1. 완료한 항목의 상태 칸을 `[x]`로 변경하라 (테이블 형식 유지)
2. 다른 항목은 절대 수정하지 마라
3. progress.md의 기존 양식(테이블 구조, 섹션 순서)을 변경하지 마라

## 절대 규칙

- **하나의 `- [ ]` 항목만 수행하고 멈춰라**. 두 번째 항목으로 넘어가지 마라.
- 작업 완료 후 다음에 할 작업이 무엇인지 한 줄로 안내하라.
- Phase 검증 기준은 해당 Phase의 마지막 항목을 완료했을 때만 실행하라.
- CLAUDE.md의 절대 규칙을 준수하라.
